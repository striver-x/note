# 一周一模式 - 享元模式

### 意图
运用共享技术有效地支持大量细粒度的对象(将大量类似对象的公共部分抽离出来并共享，减少内存开销)。

### 要解决的问题
由于使用了大量的细粒度对象而造成很多的内存开销

### 适用场景
1. 一个应用程序使用了大量的对象
2. 完全由于使用了大量对象而造成很多的存储开销
3. 对象有很多相似状态，并都可变为外部状态
